About This System

📌 System Overview

The Campus Lost & Found System is a web-based application developed to help users report, manage, and track lost and found items efficiently. It provides a simple and user-friendly interface that allows users to add new items, update item status, and search for records easily.

⚙️ Key Features
  • View real-time statistics (Total, Lost, Found, Claimed)
  • Add new lost or found items
  • Filter and search items instantly
  • Update item status (Claimed)
  • Delete records
🛠 Technology Stack
  • Frontend: HTML, CSS, Bootstrap, JavaScript
  • Backend: Node.js with Express framework
  • Database: MySQL
🔐 Security Implementation
  • Server-side validation for all form inputs
  • Parameterized SQL queries to prevent SQL Injection
  • XSS protection using input sanitization
  • Environment variables (.env) for sensitive data protection
🎯 Project Objective

The objective of this project is to develop a complete CRUD-based web application that demonstrates full-stack development skills, including frontend design, backend API development, database integration, and basic security practices.
