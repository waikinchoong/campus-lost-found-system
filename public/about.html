About This System

System Overview

The Campus Lost & Found System is a web-based application developed to help students report, manage, and track lost and found items efficiently. It provides a simple and user-friendly interface that allows users to add new items, update item status, and search for records easily.

Key Features
  • View real-time statistics (Total, Lost, Found, Claimed)
  • Add new lost or found items
  • Filter and search items instantly
  • Update item status (Claimed)
  • Edit and delete own records
Technology Stack
  • Frontend: HTML, CSS, Bootstrap, JavaScript
  • Backend: Node.js with Express
  • Database: MySQL
Security Implementation
  • Server-side validation for all inputs
  • Parameterized SQL queries to prevent SQL Injection
  • Session-based authentication
  • 404 and Global Error Handling middleware
Project Objective

This project demonstrates full-stack development skills, including frontend design, backend API development, database integration, authentication, and security implementation.
